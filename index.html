¡Bienvenido a Batalla de Códigos!

Jugá online con un código de sala. Tu secreto nunca se sube, solo se comparten los intentos.

Podés escribirlo a mano o generarlo aleatorio.
Si la sala ya existe y tiene starter, se respeta lo que ya había.
Solo vos lo ves. No se sube a la nube.
Marcá para arrancar de cero. Si te unís a una sala creada, desmarcá.

¿Cómo se juega?

Resumen rápido para nuevos:

  • Ambos eligen un código secreto de 4 dígitos (0–9). No se comparte.
  • Se turnan para adivinar el código del otro.
  • Tras cada intento, el defensor responde con “Aciertos: N” (dígitos correctos en la posición correcta).
  • Si el que empieza acierta en k intentos, el otro tiene hasta k intentos para empatar; si no llega, gana el primero; si llega, empatan.
  • Al finalizar, el ganador publica su código para que el rival vea qué tan cerca estuvo.
